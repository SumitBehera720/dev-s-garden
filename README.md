# 🍽️ Dev's Garden — Restaurant Website

A modern, minimalistic restaurant website built with the MERN stack.

## 🛠️ Tech Stack
- **Frontend:** React + Vite + Tailwind CSS v4 + Framer Motion + GSAP
- **Backend:** Node.js + Express
- **Database:** MongoDB (with in-memory fallback)

## 🚀 Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`  
Backend runs on `http://localhost:5000`

## 📁 Project Structure
```
devs-garden/
├── frontend/   # React + Vite app
└── backend/    # Express API
```
